# **Інструкція з розгортання Zabbix Server**

Цей посібник містить кроки для налаштування центрального вузла моніторингу.

## **Склад стеку**
* **Zabbix Server 7.0 (LTS)**: Основне ядро моніторингу.
* **PostgreSQL 16 + TimescaleDB**: Високопродуктивна база даних.
* **Tailscale**: Захищений VPN-тунель для зв'язку з філіями.
* **Nginx Proxy Manager**: Реверс-проксі для SSL та доступу.
